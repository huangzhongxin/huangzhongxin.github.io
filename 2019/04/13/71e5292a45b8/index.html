LeetCode题解|687. Longest Univalue Path

发表于 2019-04-13 阅读次数： Valine：

题目链接：687. Longest Univalue Path

概述

给定一棵二叉树，找出树中有相同值的最长路径长度。

思路

使用DFS，找出每个节点的左右子树左右两边的最长路径值lr，如果当前节点与左(右)子树的节点值相等，则将当前节点的左(右)值定为左(右)子树lr中的最大值+1。
